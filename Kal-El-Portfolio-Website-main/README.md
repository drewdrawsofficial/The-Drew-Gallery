# Drawings by Drew

A simple static portfolio website for artwork, drawings, portraits, paintings, and photography.

## Publish on GitHub Pages

This folder is ready to publish as a GitHub Pages static site.

Recommended setup:

1. Create a public GitHub repository named `drawings-by-drew`.
2. Upload the contents of this folder to the repository root.
3. In GitHub, open `Settings > Pages`.
4. Set the source to `Deploy from a branch`, choose `main`, and choose `/ (root)`.

The live URL will usually be:

```text
https://YOUR-GITHUB-USERNAME.github.io/drawings-by-drew/
```
